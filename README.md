# 🩺 의료용 AI 질의응답 비서 (Medical RAG QA Assistant)

🔗 **서비스 링크**: [https://medrag-6a7kjxrcdn95ghagsmh4k6.streamlit.app/](https://medrag-6a7kjxrcdn95ghagsmh4k6.streamlit.app/)

이 프로젝트는 수많은 의료 기록 데이터를 바탕으로, 여러분이 궁금해하시는 **의료 관련 질문에 똑똑하게 대답해 주는 AI 비서**입니다.

마치 의학 사전을 찾아보듯, AI가 미리 학습해 둔 방대한 의료 데이터 속에서 가장 관련 있는 내용을 찾아내어 **정확하고 믿을 수 있는 답변**을 만들어냅니다. (이러한 최신 AI 기술을 **RAG**라고 부릅니다.)
이 비서의 가장 큰 장점은 답변이 어떤 문서에서 나왔는지 **"출처(문서명 및 진료 과목)"를 항상 함께 알려준다는 것**입니다!

---

## ✨ 이 앱의 특별한 점 (주요 기능)
- **똑똑한 답변**: 의학 용어나 증상을 물어보면 저장된 실제 의료 데이터를 바탕으로 대답해 줍니다.
- **투명한 출처 제공**: AI가 임의로 지어내는 것(환각 현상)을 막기 위해, 답변의 근거가 된 문서를 꼭 표시합니다.
- **쉬운 사용법**: 웹 브라우저만 있으면 누구나 메신저처럼 쉽게 질문하고 답변을 받을 수 있습니다.
- **개인 키 사용**: 본인의 OpenAI API 키를 화면에 입력하기만 하면, 내 PC 뿐만 아니라 웹에서도 안전하게 바로 사용할 수 있습니다.

## 🛠️ 어떤 기술들이 사용되었나요?
*어려운 기술 용어들을 쉽게 풀어보았습니다.*
- **AI 두뇌 (언어 모델)**: ChatGPT로 유명한 OpenAI의 똑똑한 모델 (`gpt-3.5-turbo`)
- **데이터 검색 (임베딩 & 벡터 DB)**: 문서를 AI가 이해할 수 있는 숫자로 바꾸어 초고속으로 찾아주는 기술 (OpenAI Embeddings & FAISS)
- **전체 조정 (프레임워크)**: 여러 AI 기술들을 하나로 매끄럽게 연결해 주는 도구 (LangChain)
- **웹 화면 (UI)**: 복잡한 코드 없이도 파이썬으로 예쁜 웹 사이트를 만들어 주는 마법 같은 플러그인 (Streamlit)

## 📂 어떤 데이터를 사용했나요?
- 세계적인 데이터 플랫폼인 Kaggle의 [의료 전사 데이터셋](https://www.kaggle.com/datasets/tboyle10/medicaltranscriptions)을 사용했습니다.
- 심장내과, 이비인후과 등 다양한 진료 과목의 실제 진료 기록 샘플이 담겨 있습니다.
- 이 긴 문서들을 AI가 읽기 편하게 여러 조각으로 자르고, 빠르게 검색할 수 있도록 특수 폴더(`vectorstore`)에 저장해 두었습니다.

## 💻 내 컴퓨터에서 직접 실행해 보기

초보자도 쉽게 따라 할 수 있는 순서입니다!

### 1. 코드 다운로드 (Clone)
명령 프롬프트(터미널)를 열고 코드를 내 컴퓨터로 가져옵니다.
```bash
git clone https://github.com/wqhekjbwdsaq123/Med_RAG.git
cd Med_RAG
```

### 2. 가상 환경 만들기 (권장)
프로그램들이 서로 엉키지 않도록 이 프로젝트만의 독립된 방(가상 환경)을 만들어 줍니다.
```bash
python -m venv venv

# Windows 환경인 경우:
venv\Scripts\activate

# macOS / Linux 환경인 경우:
source venv/bin/activate
```

### 3. 필요한 도구들 한 번에 설치하기
이 앱을 구동하기 위해 필요한 재료들을 한 번에 설치합니다.
```bash
pip install -r requirements.txt
```

### 4. 프로그램 켜기!
아래 명령어를 입력하면 웹 브라우저가 열리며 앱이 실행됩니다.
```bash
streamlit run app.py
```

### 5. API 키 입력하고 대화 시작하기
웹 화면이 열리면, **왼쪽 사이드바(⚙️ Configuration)** 에 본인의 **OpenAI API Key**를 붙여넣어 주세요. 
(이제 복잡하게 설정 파일을 만들 필요 없이 화면에서 직접 입력하면 됩니다!)
키를 입력하면 시스템 준비 완료(`✅ OpenAI API Key Loaded`) 메시지가 뜨며, 하단 채팅창에 의료 질문을 던질 수 있습니다.

## 📋 폴더 구조 (어떤 파일들이 있나요?)
```text
├── app.py                  # 눈에 보이는 웹 화면과 전체 흐름을 담당하는 핵심 코드
├── requirements.txt        # 설치해야 할 파이썬 도구들의 목록표
├── vectorstore/            # AI가 검색할 의료 데이터베이스가 저장된 폴더
│   ├── index.faiss
│   └── index.pkl
└── task1_evaluation.csv    # 프로그램이 얼마나 똑똑하게 대답하는지 미리 테스트해 본 채점표
```
